NutriLab 🥗

Plataforma web de nutrição inteligente que ajuda usuários a entender sua saúde através de ferramentas interativas como cálculo de IMC, rastreamento de hidratação e cálculo de macronutrientes.

O projeto foi desenvolvido como um site moderno em HTML, CSS e JavaScript, focado em experiência visual e interatividade.

Preview

NutriLab é um Single Page Application (SPA) com:

Interface moderna e responsiva

Animações suaves

Ferramentas nutricionais interativas

Layout focado em saúde e bem-estar

Funcionalidades
📊 Calculadora de IMC

Permite ao usuário inserir:

peso

altura

idade

sexo

O sistema calcula automaticamente:

Índice de Massa Corporal

classificação (abaixo do peso, normal, sobrepeso, obesidade)

estimativa de peso ideal.

💧 Rastreador de água

Ferramenta interativa para acompanhar consumo diário de água.

Funções:

registro de copos de água

meta diária automática

cálculo em ml consumidos.

🔬 Calculadora de Macronutrientes

Calcula macros ideais com base em:

peso corporal

nível de atividade

objetivo (perder peso, manter ou ganhar massa).

Resultado:

proteínas

carboidratos

gorduras

calorias recomendadas.

🍽 Receitas saudáveis

O site apresenta receitas com:

tempo de preparo

calorias

quantidade de proteína

categorias como:

vegano

low carb

high protein.

📚 Dicas nutricionais

Sessão educacional com hábitos saudáveis como:

hidratação

consumo de proteínas

variedade alimentar

qualidade do sono.

Tecnologias usadas

HTML5

CSS3

JavaScript Vanilla

Google Fonts

Não utiliza frameworks ou backend.
